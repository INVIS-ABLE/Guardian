# Security Testing Calendar

> Status: draft — owner TBD. Part of Guardian governance (docs/governance/).

Cadence of tests, drills and reviews.

- Per PR: unit/property/policy tests, scanners (blocking)
- Weekly: scheduled scans, Scorecard
- Monthly: AI red-team (PyRIT) when models/prompts/tools change; Sigma rule validation
- Quarterly: DR restore exercise, key-rotation drill, chaos tests
- Annually: independent security review + authorised penetration test
